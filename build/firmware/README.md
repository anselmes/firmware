# Firmware

U-Boot firmware will be copied here

Example:

```text
coreboot/u-boot.rom
coreboot/u-boot.bin
coreboot/u-boot.dtb
coreboot/u-boot.elf
```

## Building

Example:

```bash
cmd/run.sh uboot build coreboot
```
